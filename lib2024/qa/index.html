  5. Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

  7. What element is used to create images? Name it's attributes

    Answer:An element is everything within a tag. The image tag is <img>. The full element appears as: <img src="img/image-1.jpg" alt="Image 1" width="300" height="200"> where the attribute src indicates the image source or file name, the alt attribute is the text which displays when the image doesn't load, and the width and height attributes indicate the desired size.
